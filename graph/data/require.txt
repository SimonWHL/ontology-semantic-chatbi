1.1 节点类型

| 节点类型 | 含义 | marketing_cn 实例 | 物理落地映射 |
|---------|------|------------------|-------------|
| **Entity（实体）** | 业务对象 | 商机、客户、销售雇员、城市、业务中心、产品、合同、订单 | cube 的 primary/foreign/unique entity（`opportunity_id`、`business_center`、`buss_permission`） |
| **Event（事件）** | 改变实体状态、影响指标的动作 | 商机创建、预计签单、中标、首次签章、首次下单、首次出库、发货/出库 | 日期/状态列（`create_time`、`expected_sign_date`、`is_won_bid`、`first_outbound_date`、`opportunity_status`） |
| **Attribute（属性/维度）** | 实体的描述性属性 | 商机类型/状态、行业/子行业、产品类别、BG、业绩地市 | `cube_xxx.dim_yyy` |
| **Metric（指标）** | 可聚合度量 | 销售额、月任务目标、业绩缺口、业绩达成率、去重商机金额、交货总金额、出库额、未转合同金额、未清订单金额 | `metric_ref`（simple/derived） |
| **Concept/Filter（业务口径）** | 业务语言定义的筛选概念 | 赢单、中标、在途、未中标、规上(≥10万)、潜力产品、大项目、PBG/EBG、本期/非本期/本年预签 | `filter_ref` / `tsf_*` |
| **Rule（规则）** | 权限/安全/默认推理 | 城市权限门控、行级安全、默认推理(缺时间→本财年、同比→年同比) | `ontology/business/{rules,gate_rules}`、`默认推理规则.md` |
| **Function（语义函数）** | 时间/分析函数 | 本期、本财年、近N月、同环比、占比、排名 | `tsf_*`、`indirection` |
| **SubjectDomain（主题域）** | 场景分区 | performance / opportunity / sales_delivery / permission | `subject_domains/cn_market_subject.yaml` |
| **AnalysisView（分析视图）** | 用户可直接问的复合分析概念，聚合多条查询路径（类比 dashboard） | 经营情况、业绩情况 | 见 §1.4，`ontology/analysis_views/*.yaml` |

1.2 边类型

| 边 | 方向 | 含义 | 示例 |
|----|------|------|------|
| `has_attribute` | Entity → Attribute | 实体的属性 | 商机 → 商机状态 |
| `has_event` | Entity → Event | 实体经历的事件 | 商机 → {创建, 预计签单, 中标, 首次出库} |
| `state_transition` | Event → Event | 事件状态流转 | 创建 → 预计签单 → 中标 → 首次出库 |
| `relates_to` | Entity ↔ Entity | 实体间关联（替代 cube join） | 商机 —(opportunity_id)→ 出库明细 |
| `measured_by` | Entity/Event → Metric | 被指标度量 | 出库事件 → 出库总额 |
| `dimensioned_by` | Metric → Attribute | 指标可按某属性分组 | 销售额 → 产品类别 |
| `derived_from` | Metric → Metric/Function | 衍生指标依赖（同环比/占比/排名） | 业绩达成率 = 销售额/月目标；销售额占比 ← proportion_by |
| `constrains` | Concept → Entity/Attribute/Metric | 口径约束 | 中标 → 商机状态；规上 → 金额≥10万 |
| `governs` | Rule → Entity/Metric/AnalysisView | 规则管控 | 城市权限规则 → 经营情况 |
| `belongs_to` | 任意节点 → SubjectDomain | 主题域归属 | 去重商机金额 → opportunity |
| `alias_of` | 节点 → 同义词 | 自然语言别名 | 业绩地市 ← {城市, 地市} |
| **`decomposes_to`** | **AnalysisView → 查询路径** | **分析视图分解出的（有序）查询路径**，边带 `order` 属性 | 经营情况 → {路径1, 路径2, …, 路径13} |
