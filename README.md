# 基于本体语义网络的智能问数

> 本体化语义层驱动的 Data Agent 智能数据分析系统

## 项目简介

本项目探索基于本体语义网络构建智能问数系统的技术路径，核心方向包括：

- **本体语义层架构**：三层语义体系（指标语义层 → 企业世界语义层 → 动态语义层）
- **知识工程方法论**：知识原子化、四层知识结构、来源标签与信任链
- **AI Agent 能力评估**：五层能力模型（语义理解 / 任务推理 / 执行行动 / 自省认知 / 信任治理）

## 技术路线

```
自然语言 → 语义理解 → LogicForm（中间表示）→ SemanticDB（语义数据库）→ SQL/API/Agent动作
```

## 知识库结构

```
├── 语义层架构/          # 本体语义理论体系
├── 知识工程方法论/       # 个人知识工程结构化方法
├── AI_Agent能力评估/    # Agent 能力评估框架
├── 原文存档/            # 技术文章原文
└── docs/               # 项目文档
```
